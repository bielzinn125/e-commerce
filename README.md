# ECommerce

Este projeto foi gerado usando [Angular CLI](https://github.com/angular/angular-cli) version 22.0.4.

## Servidor de desenvolvimento

Para iniciar um servidor de desenvolvimento local, execute:

```bash
ng serve
```

Uma vez que o servidor estiver rodando, abra seu navegador e vá para `http://localhost:4200/`. O aplicativo vai recarregar automaticamente sempre que você modificar algum dos arquivos de origem.

## Estrutura de código

O Angular CLI inclui ferramentas poderosas de scaffolding de código. Para gerar um novo componente, execute:

```bash
ng generate component component-name
```

 Para uma lista completa de esquemas disponíveis (como `components`, `directives`, ou `pipes`), execute:

```bash
ng generate --help
```

## Construção

Para construir o projeto, execute:


```bash
ng build
```
Isso vai compilar seu projeto e armazenar os artefatos de build no diretório `dist/`. Por padrão, o build de produção otimiza sua aplicação para desempenho e velocidade.

## Executando testes unitários

Para executar testes unitários com o executor de testes [Vitest](https://vitest.dev/), use o seguinte comando:

```bash
ng test
```

## Executando testes de ponta a ponta

Para testes de ponta a ponta (e2e), execute:

```bash
ng e2e
```

O Angular CLI não vem com um framework de teste de ponta a ponta por padrão. Você pode escolher um que atenda às suas necessidades.
